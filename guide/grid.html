# 响应式布局

同时设置多个配置参数，实现响应式布局

点击查看代码 
<d-row class="demoRow">
  <d-col span="24" :ipad="{span:12}" :narrow-pc="{span: 6}" :pc="{span: 3}">
    <div class="bg-purple-dark">内容1</div>
  </d-col>
  <d-col span="24" :ipad="{span:12}" :narrow-pc="{span: 6}" :pc="{span: 3}">
    <div class="bg-purple">内容2</div>
  </d-col>
  <d-col span="24" :ipad="{span:12}" :narrow-pc="{span: 6}" :pc="{span: 3}">
    <div class="bg-purple-light">内容3</div>

  <d-col span="24" :ipad="{span:12}" :narrow-pc="{span: 6}" :pc="{span: 3}">
    <div class="bg-purple">内容5</div>

# d-row 属性

参数 说明 类型 可选值 默认值
gutter 栅格的间隙，单位为 px String , Number —— ——
offset 举例左边的举例，单位为 px String , Number —— ——

# d-col 属性

参数 说明 类型 可选值 默认值
span,ipad,narrow,pc 不同屏幕宽度下分栏所占宽度的配置参数，键对应分栏所占宽度，值对应分栏前空隙宽度。span,ipad,narrow,pc,widePc对应 min-width 分别为 577px、769px、993px、1201px Object —— ——
