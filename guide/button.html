# button 按钮

基础组件，点击触发业务逻辑。

# 基础用法

点击查看代码 
<d-button >default</d-button>
<d-button primary>primary</d-button>
<d-button error>error</d-button>
<d-button icon="settings">default</d-button>
<d-button icon="settings" position="right">default</d-button>
<d-button :loading="true" >loading</d-button>

# 属性

参数 说明 类型 可选值 默认值
icon icon 图标名 String —— ——
position icon 位置 String left , right left
primary 主题色按钮 Boolean —— false
error 警告色按钮 Boolean —— false
loading 带 loading 效果的按钮 Boolean —— false
