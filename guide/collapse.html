# Collapse 折叠面板

将内容区域折叠/展开。

# 基础用法

点击查看代码 
<h4>默认，多item展示</h4>
<d-collapse :selected.sync="selected">
  <d-collapse-item name="new" title="风闻">在原油宝亏损1000万的详细经历</d-collapse-item>
  <d-collapse-item name="internation" title="国际">武汉P4实验室主任接受路透社采访，驳斥阴谋论</d-collapse-item>
  <d-collapse-item name="military" title="军事">美舰闯西沙，我军警告驱离</d-collapse-item>
  <d-collapse-item name="finance" title="财经">多家银行暂停纸原油及其他相关业务</d-collapse-item>
</d-collapse>
<h4>单item展示</h4>

<d-collapse single :selected.sync="selected">
  <d-collapse-item name="new" title="风闻">在原油宝亏损1000万的详细经历</d-collapse-item>
  <d-collapse-item name="internation" title="国际">武汉P4实验室主任接受路透社采访，驳斥阴谋论</d-collapse-item>
  <d-collapse-item name="military" title="军事">美舰闯西沙，我军警告驱离</d-collapse-item>
  <d-collapse-item name="finance" title="财经">多家银行暂停纸原油及其他相关业务</d-collapse-item>
</d-collapse>

# d-collapse 属性

参数 说明 类型 可选值 默认值
selected 当前展开 item 的 name 组成的数组。可以用 .sync 修饰符绑定。 Array —— ——
single 只展示一个item Boolean —— ——

# d-collapse-item 属性

参数 说明 类型 可选值 默认值
name item 的名称，唯一且必填 String —— ——
title item 标签名 String —— ——
