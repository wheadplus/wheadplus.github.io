# slides 轮播

在有限的空间内循环播放图片、文字等内容。

# 基础用法

点击查看代码 
<d-slides class="wrapper"  :selected.sync="selected">
  <d-slides-item name="1">
    <div class="demo-box">1</div>
  </d-slides-item>
  <d-slides-item name="2">
    <div class="demo-box">2</div>
  </d-slides-item>
  <d-slides-item name="3">
    <div class="demo-box">3</div>
  </d-slides-item>
</d-slides>

data() {
  return {
    selected: '1'
  }
}


// css
.demo-box {
  height: 300px;
  line-height: 300px;
  text-align: center;
  font-size: 24;
  background: cornflowerblue;
  color: #fff;
}

# d-slides 属性

参数 说明 类型 可选值 默认值
selected 当前选中滚动item String —— ——
autoPlay 开启自动轮播 Boolean —— true
autoPlayDelay 停留时间开始滚动，默认停留3000毫秒 Number —— 3000

# d-slides-item 属性

参数 说明 类型 可选值 默认值
name item名字，通过name识别当前item，必填 String —— ——
