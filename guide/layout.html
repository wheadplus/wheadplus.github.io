# layout 容器布局

容器布局，可快速搭建页面的基本结构，需浏览器支持flex布局。在使用容器布局时，layout组件的子组件只能是header，content，footer，sider或layout，同时，上述子组件的父组件也只能是layout。

# 普通上中下三栏布局

点击查看代码 
<d-layout style="color: white; margin-bottom:50px;">
  <d-header class="demo-center" style="height: 50px; background:lightskyblue;">header</d-header>
  <d-content class="demo-center" style="height: 100px; background:deepskyblue;">content</d-content>
  <d-footer class="demo-center"  style="height: 50px; background:lightskyblue;">footer</d-footer>
</d-layout>

# 上中下三栏布局，主体部分带侧边栏

点击查看代码 
<d-layout style="color: white; overflow:hidden; margin-bottom:50px;">
  <d-header class="demo-center" style="height: 50px; background:lightskyblue;">header</d-header>
  <d-layout>
    <d-sider class="demo-center" style="height: 100px; background:#ddd; width:200px; color: black;">sider</d-sider>
    <d-content class="demo-center" style="height: 100px; background:deepskyblue;">content</d-content>
  </d-layout>
  <d-footer class="demo-center" style="height: 50px; background:lightskyblue;">footer</d-footer>
</d-layout>
